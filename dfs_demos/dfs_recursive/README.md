# dfs_recrusive

### Environment

- C++ 20

### Examples

- [AtCoder](https://atcoder.jp/contests/atc001/tasks/dfs_a)

題目有`s`作為起點，`g`作為終點，`#`作為牆壁，需要判斷`s`是否能走到`g`。從`s`開始移動，因為不能斜角移動，所以只需要對4個方向(上、下、左、右)進行探索便可。
